# Data Analysis Project: 

This repository contains a clinical dataset and R scripts for performing survival analysis. 

## Files and Folders

- **Clinical_Data_Validation_Cohort.xlsx**  
  The main dataset containing clinical information for 95 patients, including survival time, event status, tumor grade, stage, sex, smoking status, adjuvant therapy type, and mutation status.

- **Survival_Analysis/**  
  Folder containing all R scripts and outputs for performing survival analysis:
  - `Survival_Analysis.R`: R script performing Kaplan-Meier survival curves and Cox regression analysis.
  - `Overall_Survival.png`, `Survival_by_Sex.png`, `Survival_by_Grade.png`, `Survival_by_Therapy.png`: Kaplan-Meier survival plots.
  - `Survival_Analysis.pdf`: A compiled PDF report of the analysis.
